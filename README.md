# 🎙️ Smart議事録

> Gemini API を使って会議音声を書き起こし、「聞くことリスト」への自動チェック・AI要約・議事録生成までを自動化する Windows 向けデスクトップアプリです。

## ✨ 主な機能

- **書き起こし** — Gemini API がマイク入力をテキスト化
- **聞くことリスト** — 最大30問の確認事項をあらかじめ設定し、AI が自動チェック
- **AI 自動要約・決定事項抽出** — 会議終了後に概要・決定事項・アクションアイテムを構造化して生成
- **テンプレート管理** — 定例会議の聞くことリストをテンプレート保存してワンクリック呼び出し
- **Markdown エクスポート** — 要約・チェック結果・全文書き起こしをまとめてファイル出力
- **プロンプトカスタマイズ** — AI に渡すプロンプトを設定画面から自由に編集可能


## 🔒 プライバシー

音声ファイル・議事録データはすべて **PC ローカルにのみ保存**されます。
書き起こしや要約時にGeminiAPIサーバーへの通信が発生します。


## 🚀 始め方

1. [Google AI Studio](https://aistudio.google.com/) で Gemini API キーを取得（クレジットカード不要）
2. リリースページから最新の `.exe` インストーラーをダウンロード
3. アプリの設定画面に API キーを貼り付けて完了

## 動作環境

- Windows 10 / 11
- Gemini API キー（Google AI Studio で無料取得可）
